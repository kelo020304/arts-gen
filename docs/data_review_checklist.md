# 数据质量审查 checklist

> 范围: 检查每批**需要数据方人工改**的字段 / 几何问题。
> 最后更新: 2026-05-06


| 文件              | 字段           | 检查点                                                                                                                                                                                 | 已知问题样例                                                                                                                                                                                                                    |
| ----------------- | -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `model_info.json` | `up axis`      | 是不是真实的轴向，不能标`y-up` 实际是 `z-up`                                                                                                                                           |                                                                                                                                                                                                                                 |
| `{uuid}.json`     | `object_name`  | 是不是物体的真实名字                                                                                                                                                                   | `01c6cde4` 标 `"Book"`，实际是浓缩咖啡机                                                                                                                                                                                        |
| `{uuid}.json`     | `category`     | 是否对得上`object_name`、有没有拼错                                                                                                                                                    | `01568948` 拼成 `"Hou Ehold"`；`01c6cde4` 写 `"Book"`                                                                                                                                                                           |
| `{uuid}.json`     | `parts[].name` | 每个 part 都要有描述性名字：不能用占位词、不能多个 part 重名、不能拼错。**首选功能名**（如 `"Power Button"`）；不知道功能时退而用方位名（如 `"Left Button"`），但不能停留在 `"Button"` | 占位：`"Part"`；重名：3 个按钮全叫 `"Button"`，应改成 `"Power Button"` / `"Brew Button"` / `"Steam Button"`，实在不知道功能就 `"Left Button"` / `"Middle Button"` / `"Right Button"`；拼错：`"Team wand 2"` 应是 `"Steam Wand"` |
