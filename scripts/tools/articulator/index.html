⑤ Sites — 语义 AABB

          

            Site = 不可见的 AABB 盒, 标注 part 内部的语义区域 (屏幕 / 按键 / 摄像头 / 把手).

            导出后变成 /<part>/Sites/<site_id> 的 invisible Cube,

            渲染器跳过, 但 robot policy 可以查询位姿 (随 articulation 自动跟随).


            怎么标: 在右侧每个 part 卡片里点 + 框选 site 后, 在 viewport 拖一个矩形 — 框内 part 顶点的 AABB 就是这个 site.
